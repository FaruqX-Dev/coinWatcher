CoinWatcher is a clean, lightning-fast cryptocurrency tracker built to help investors monitor the market without the noise. As a solo developer project, it focuses on performance, local-first state management, and real-time data accuracy.

## üöÄ Features

- **Live Market Data**: Real-time tracking of 10,000+ coins via the CoinGecko API.
- **Personal Watchlist**: Securely save favorite coins to your profile, synced across devices via Firebase Firestore.
//- **Smart Push Notifications**: Receive alerts for significant 24h price percentage changes.
- **Deep Insights**: Market cap rankings, total volume, and historical price stats.
- **Secure Auth**: Seamless login/signup using Firebase Authentication.
